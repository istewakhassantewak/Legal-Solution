⚖️ Legal Solution

🔗 Live Site: https://legal-solution-two.vercel.app

A modern and responsive legal service landing page built using HTML and Tailwind CSS. This project focuses on clean UI, accessibility, and a professional layout suitable for law firms or legal consultants.

🚀 Features
⚖️ Elegant and professional legal service design
📱 Fully responsive across all devices
📄 Sections for services, client feedback, FAQ, and contact
❓ Interactive FAQ section
📞 Contact form and information section
🎨 Styled using Tailwind CSS utility classes

🛠️ Technologies Used
HTML5
Tailwind CSS
DaisyUI (optional if used)

📁 Project Structure
├── images/             # Image assets
├── index.html          # Main HTML file
├── tailwind.config.js  # Tailwind configuration
├── README.md           # Project documentation
└── .gitattributes
⚙️ Setup & Usage

Clone the repository:

git clone https://github.com/istewakhassantewak/legal-solution.git

Navigate to the project folder:

cd legal-solution
Open index.html in your browser.
